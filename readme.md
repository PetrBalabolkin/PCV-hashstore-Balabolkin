# PCV-hashstore-Balabolkin

## Getting Started

### 1. Clone repo

```bash
git clone https://github.com/PetrBalabolkin/PCV-hashstore-Balabolkin
cd PCV-hashstore-Balabolkin
```

### 2. Install dependencies

Use one of the following package managers to install the dependencies:

```bash
pnpm install
# alebo
bun install
# alebo
yarn install
# alebo
npm install
```

### 3. Build project

```bash
pnpm build
# alebo
npm run build
```

### 4. Start the CLI

```bash
cd dist
node cli.js
# alebo
bun cli.js
```
